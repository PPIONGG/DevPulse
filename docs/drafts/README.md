# Drafts

Feature drafts ที่อยู่ระหว่างคุยกัน — เน้นว่า module ทำอะไร ฟีเจอร์อะไรบ้าง UI หน้าตาเป็นยังไง

พอ draft ตกลงกันแล้ว จะย้ายไปเป็น spec เต็ม (พร้อม SQL, Go, TypeScript) ใน `docs/modules/`

## Flow

```
1. สร้าง draft ใน docs/drafts/     → คุยกันว่าจะทำอะไร
2. confirm แล้ว                     → เขียน spec เต็มใน docs/modules/
3. implement                        → เขียนโค้ดตาม spec
```
